### 📝 Description
A Python-based **Student Management System** designed to help educational institutions manage and organize student records efficiently.

---

## 🚀 Features
* **➕ Add Student** – Quickly register new students with their details.
* **👁️ View Student** – Display a clean list of all registered students.
* **🔄 Update Student** – Modify existing student records seamlessly.
* **❌ Delete Student** – Remove outdated or incorrect student profiles.
* **🔍 Search Student** – Instantly find specific students using search filters.
